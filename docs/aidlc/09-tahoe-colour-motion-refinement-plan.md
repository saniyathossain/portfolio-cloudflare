### D. Signature / uncommon motion (all four)  (`styles.css` + `index.html` classes + `motion.js`)

2. **Panel cursor spotlight** — reuse the existing `.spec` mechanism (`motion.js` sets `--px/--py`; CSS at
   `styles.css:258`). Add `.spec` to the glass panels (`services-list`, each skills `.glass-panel`,
   `stats-panel`) in `index.html`; add a panel-sized radial variant so the highlight tracks across the whole
   card. Confirm `motion.js` `specular()` binds all `.spec` (it queries the class → auto-applies).
3. **Aurora parallax** — in `motion.js`, add a throttled scroll handler (rAF) that drifts the aurora orbs
   (`body::before` / hero glow) and applies a gentle scroll-tied `hue-rotate`/translate for living depth.

4. **Sheen sweep on hover** — a `.sheen` utility: a skewed light-gradient `::before` that glides across on
